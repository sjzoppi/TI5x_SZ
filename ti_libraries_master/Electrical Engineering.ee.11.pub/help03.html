EE-03 — S x⇌t Y Parameter Conversions

Small-signal, two-port "black boxes" are often characterized in terms of complex scattering (S) parameters normalized to a common input and output transmission line impedance Z0. The translation of these parameters to admittance (Y) parameters is necessary to create equivalent circuit models. This program converts S parameters to Y parameters or vice versa.

Y11Y12
Y21Y22

=

1-S11+S22-ΔS / C-2S12/C
-2S21/C1-S11-S22-ΔS / C

Where ΔS = S11 S22 - S12 S21 and S parameters are input in polar form (R, θ).

And C = 1 + S11 + S22 + ΔS

When converting from S to Y parameters, once the matrix elements are calculated, the U matrix is multiplied by the real scalar Z0-1 to give the actual Y parameter.

When converting from Y to S parameters, the Y parameters are multiplied by Z0 before the matrix elements are calculated to give the actual S parameters.

  • Initialize program :  SBR   CLR 
  • Enter polar form of S (or Y) matrix
    • Magnitude S11 (or Y11), press  x⇌t 
    • Angle S11 (or Y11), press  A 
    • Magnitude S12 (or Y12), press  x⇌t 
    • Angle S12 (or Y12), press  B 
    • Magnitude S21 (or Y21), press  x⇌t 
    • Angle S21 (or Y21), press  C 
    • Magnitude S22 (or Y22), press  x⇌t 
    • Angle S22 (or Y22), press  D 
  • Enter real characteristic impedance in ohms and press one of the following:
    • Convert S -> Y, press  E 
    • Convert Y -> S, press  2nd   E' 
  • Display conversion results
    • Magnitude Y11 (or S11), press  2nd   A' 
    • Angle S11 (or Y11), press  x⇌t 
    • Magnitude Y12 (or S12), press  2nd   B' 
    • Angle S12 (or Y12), press  x⇌t 
    • Magnitude Y21 (or S21), press  2nd   C' 
    • Angle S21 (or Y21), press  x⇌t
