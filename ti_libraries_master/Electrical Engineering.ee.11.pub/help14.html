This program computes component values for passive lowpass Butterworth and Tchebycheff filters of the following form.

Required inputs are the filter order n (n, an integer ≥ 1) which defines the number of reactive components, allowable ripple ε in dB (ε = 0 for Butterworth filters), terminal resistance R in ohms, and the cutoff frequency fc in hertz.

NOTE: This program assumes that the generator resistance Rg is equal to the load resistance RL for Butterworth filters and odd-order Tchebycheff filters. Thus, the termination resistance R = Rg = RL. In the case of even-order Tchebycheff filters R>L = R>g / coth2 (β /4) which, for small ripple results in the generator resistance Rg being approximately equal to the load resistance RL. Thus, the termination resistance R = Rg ≈ RL.

  • Initialize program :  SBR   CLR 

    Enter the following in any order

    • Order of the filter, n, press  A 
    • Allowable ripple in dB, ε, press  B 
    • Termination resistance in ohms, R, press  C 
    • Cutoff frequency in hertz, fc, press  D 
  • Compute and display component values. C is in farads. L is in henry's, press  2nd   E' , display 1, C1, then press  R/S  to display 2, L2, then press  R/S  to display 3, C3...

    OR

  • Compute and print component values, press  D  which print the same values as above.
