This program transforms a real time series to the frequency domain (DFT) and performs the inverse transform (IDFT) from the frequency domain back to the time domain. The frequency representation of the transform is given in terms of magnitude in dB and phase in degrees. The program allows a DFT sampling rate of up to 88 points. For IDFT's, the user can generate a time-domain image of a signal by taking up to 32-point sample of its symmetric frequency domain representation. It is assumed that the time series reflects a periodic function over some finite interval so that the frequency domain behavior can also be expressed as a discrete series representing a periodic function.

The program assumes that the time function. f and the frequency function F are approximated by N equally spaced samples at intervals of ΔT (DFT) or 1/ΔT (IDFT). These N samples define the Discrete Fourier Transform pair f ⇔ F because the N time and frequency values give a discrete representation of the time and frequency domain waveforms, respectively. To minimize the effects of aliasing, choose N such that the DFT sample interval is ΔT≤1/2fc where fc is the highest frequency component of the transform and the IDFT sample interval is 1/ΔT≥2fc which is the Nyquist sampling rate.

Data Input

Data is input has described in the usage instruction below. Alternatively the data can be input directly into the following data registers.

For DTF: f(0) → R12
f(1) → R13
:
f(n) → Rn + 12
:
f(87) → R99

For IDTF: Fre(0) → R12 ; Fim(0) → R44
Fre(1) → R13 ; Fim(1) → R45
:
Fre(n) → Rn + 12 ; Fim(n) → Rn + 44
:
Fre(31) → R43 ; Fim(31) → R75

This program is designed for use with the printer.

  • Initialize program :  SBR   CLR 

    For DTF

  • Enter sample number (0 ≤ n ≤ N - 1), press  A 
  • Enter sample value, press  B 

    Repeat both steps above until all data is entered.

  • Enter total number of samples N, press  B  to compute DFT. Values displayed are:
    • sample number
    • real part of transform,
    • imaginary part of transform
    • magnitude of transform
    • magnitude of transform in dB
    • phase of transform in degrees

    This six-part sequence is repeated for all values of n.

    For IDTF

  • Enter sample number (0 ≤ n ≤ N - 1), press press  A 
  • Enter magnitude of transform at n, press  C 
  • Enter phase in degree of transform at n, press  D 

    Repeat the three steps above until all data is entered.

  • Enter total number of samples N, press  2nd   E'  to compute IDFT. Values displayed are:
    • sample number
    • time value at n

    This two-part sequence is repeated for all values of n.
