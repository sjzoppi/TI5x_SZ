ML-07 — Polynomial Evaluation

This program evaluates a polynomial given the degree n, the n + 1 coefficients a0 .. an, and the argument x. Once the degree and coefficients have been entered, the polynomial may be evaluated for any number of different x values.

First, enter the integer n and press  A .

Then, enter the starting power (normally 0), and press  B  to start entering coefficients; it will show the next power, and after you have entered each coefficient, press  R/S  to continue. When you see -n being displayed, you know you’re done.

If you make a mistake entering any coefficients, simply press  RST  to clear the pending subroutine return, put in the starting coefficient number from which you want to re-enter, press  B , and continue from there, entering coefficients and pressing  R/S  after each one as before.

Once all the coefficients have been entered, simply enter a value x and press  C  to evaluate the polynomial for that value.

Formula Used

The polynomial is evaluated using Horner’s Rule, which can be expressed recursively as follows:

P(x) = P(x, n)

where

P(x, 0) = an

P(x, i) = x × P(x, i - 1) + an - i for i > 0

Registers used: 01-05+n.
