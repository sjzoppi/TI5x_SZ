ML-08 — Zeroes Of Functions

This program finds values of x for which a user-defined function f(x) is zero (to within a user-specified tolerance), within a specified interval. It does this by sampling values of the function at a specified step interval, looking for successive points where the function changes sign; then it narrows down the location of the crossing point to within the specified tolerance.

  • Switch to  2nd   Pgm  00 and enter program steps to compute the desired function f(x), beginning with  2nd  Lbl  2nd  A´ and ending with  INV   SBR . Do not use  = ,  CLR ,  RST  or registers 01-09. This function will be invoked with x in the display register, and should leave f(x) there when it returns.
  • Switch back to  2nd   Pgm  08 for the remaining steps.
  • Enter the lower bound of the interval over which to evaluate the function, and press  A .
  • Enter the upper bound of the interval over which to evaluate the function, and press  B .
  • Enter the step size at which to sample the function while looking for crossing points, and press  C . Roots closer together than this are liable to be missed.
  • Enter the tolerance by which the function result can differ from zero when narrowing down to a root, and press  D . If this is not done, a default tolerance of 0.01 will be assumed.
  • Finally, press  E  to search for a root.
  • The program will stop when a root is found, or the search has reached its end. If a value is displayed without any error indication, it is a root; press  E  or  R/S  to resume the search for the next root. If the number is flashing, then there are no more roots to be found.

Registers used: 01-07, 09.
