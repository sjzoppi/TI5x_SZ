ML-09 — Simpson’s Approximation (Continuous)

This program computes the numerical definite integral of a given function over a given interval, to an accuracy determined by a given number of subdivision steps. The function is defined by a subroutine in the user program memory.

  • Switch to  2nd   Pgm  00 and enter program steps to compute the desired function f(x), beginning with  2nd  Lbl  2nd  A´ and ending with  INV   SBR . Do not use  = ,  CLR ,  RST  or registers 01-06. This function will be invoked with x in the display register, and should leave f(x) there when it returns.
  • Switch to  2nd   Pgm  09 for the remaining steps.
  • Enter the lower bound x0 over which to evaluate the integral, and press  A .
  • Enter the upper bound xn over which to evaluate the integral, and press  B .
  • Enter the number of steps n into which to divide the interval and press  C . This must be a positive even integer, otherwise the display will flash.
  • Finally, evaluate the integral by pressing  D . When this finishes, the result should be displayed.

Registers used: 01-06.
