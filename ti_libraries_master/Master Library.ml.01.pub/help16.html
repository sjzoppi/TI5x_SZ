ML-16 — Combinations, Permutations And Factorials

Given a non-negative integer n and a non-negative integer r ≤ n, this program can compute n! as well as P(n, r) and C(n, r).

First, enter n and r, in that order:

  1. n  A 
  2. r  B 

Now you can compute any of the available functions on n and r:

  • n factorial:  C 
  • Number of permutations of r objects out of n:  D 
  • Number of combinations of r objects out of n:  E 

Formulas Used

Factorial can be defined recursively as follows:

0! = 1

n! = n × (n - 1)! (for n > 0)

Numbers of permutations and combinations are calculated in terms of factorials as follows:

P(n, r) = n! ÷ (n - r)!

C(n, r) = n! ÷ ((n - r)! × r!)

Registers used: 01-04.
