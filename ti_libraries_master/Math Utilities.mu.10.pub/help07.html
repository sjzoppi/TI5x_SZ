MU-07 — Data Arrays

The handling of arrays or matrices of data is always a time consuming job. This program can simplify handling of data arrays which have a large amount of fixed data and only a percentage of the data items change from time to time.

The first requirement of this program is that you set the size of the array. If you have a table of 3 x 5 values you need to specify 4 x 6 to account for the last row and column used for results of some sort.

Note that rows or columns numbers specified in the decimal part must have 2 digits. So values 1 to 9 must have a leading 0.

Entering Data

Entering Data

  • Enter total number of rows and columns in the form R.C, press  2nd   B' . Note that a 4x6 array as 4.06.
  • To cancel last row/column isolation automatically set by the step above, press  2nd   St Flg   0 

Entering Array Data

  • Enter row number of data to be entered, press  2nd   C' 
  • Enter data items column by column in left to right order, press  C . Repeat those steps until all data are entered.

Entering Linear Growth Data

  • Enter row number of data to be entered, press  2nd   C' 
  • Enter first data item in row, press  C 
  • Enter growth rate as decimal percent, press  D 

Entering Single Data items

  • Enter row and column number of data item, press  2nd   C' 
  • Enter data item, press  C 

Data Array Computations

Row and Column Totals

  • Compute the total of a row, enter row number and press  E 
  • Store row total in last column of row, press  C 
  • Compute the total of a column, enter column number and press  +/-   E 
  • Store column total in last row of column, press  C 

Row and Column Products

  • Compute the product of data items in a row, enter row number and press  2nd   E' 
  • Store product of row in last column of row, press  C 
  • Compute the product of data items in a column, enter column number and press  +/-   2nd   E' 
  • Store product of column in last row of column, press  C 

RowA [] RowB = RowC, where [] is one of + - x ÷ yx

  • Enter numbers of rows A and B in the format A.B and press  A 
  • Enter number of row C, press  C 
  • Compute RowA [] RowB, press  SBR  []

RowA [] Constant = RowC

  • Enter constant value, press  2nd   A' 
  • Enter numbers of rows A, press  A 
  • Enter number of row C, press  B 
  • Compute RowA [] Constant, press  SBR  []

Constant [] RowB = Row C

  • Enter constant value, press  2nd   A' 
  • Enter numbers of rows B, in the for .B press  A 
  • Enter number of row C, press  B 
  • Compute RowA [] Constant, press  SBR  []

Shift Right Row A, result in RowB

  • Enter numbers of rows A and B in the format A.B and press  A 
  • Enter position to be shifted, press  2nd   D' 

Shift Left Row A, result in RowB

  • Enter numbers of rows A and B in the format A.B and press  A 
  • Enter position to be shifted, press  +/-   2nd   D' 

Recall Array Data

  • Enter row number, recall first data item, press  +/-   B 
  • Recall next data item in row, press  C 
  • To print all data items in a row, press  +/-   B 

Registers: 00 - 06, 07 and up for data
