MU-08 — Data Packing

The effective number of available data registers can be increased by packing data. Using this program, each register starting with R04 is divided into a selected number of smaller registers as specified by a single format statement. For example, the following statement divides each data register into three pseudo registers and specifies the length of each pseudo register.

3.526

  • 3 pseudo registers per data register
  • 5 digits for pseudo register 1
  • 2 digits for pseudo register 2
  • 6 digits for pseudo register 3

The format is flexible within the limits of 1 to 9 pseudo registers per data register. Maximum length of a pseudo register is 9 digits and the maximum combined length for all pseudo registers in a data register is 13 digits. (only 11 digits in this implementation).

The program handles only integer values.

  • Enter format, press  A 
  • Then enter data:
    • Enter data, press  x⇌t 
    • Enter pseudo register number, press:
      • to store :  B 
      • to recall :  C 
      • to exchange :  D 

Registers: 00-03
