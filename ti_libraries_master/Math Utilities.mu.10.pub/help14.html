This program fits and (n -1)th order polynomial to n input data points using Aitken's Method. The resulting polynomial is then evaluated at an intervening point (x0) to predict the value of f(x0).

Use as a Subroutine

User must store data in the contiguous data registers in the order x1, x2, ... xn ; y1, y2, ... yn ; plus n additional working registers following yn. User must also place address of x1 in R10, n in R11, and x0 in R12. Value of f(x0) will be in the display upon return from the subroutine.

  • Enter address of x1, press  A 
  • Enter number of (x,y) pairs, press  2nd   A' , then for each pair:
    • Enter x value, press  B 
    • Enter y value, press  C 
  • Enter x0 and calculate f(x0), press  D 

Registers used: 8-17, and see above.
