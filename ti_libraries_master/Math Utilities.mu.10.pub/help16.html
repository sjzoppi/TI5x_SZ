MU-16 — MINIMAX

Given a function f(x), this program finds those values of x (called critical values) for which f(x) has a maximum or a minimum. To run this program, you must first enter the function f(x) as a series of keystrokes. Then you must enter a beginning value, x0, and a positive search increment, Δx.

Search is always performed in the positive direction and is begun at the point x0 + Δx. You can enter a limit on the number of iterations the program will go through to locate a critical point.

Once a critical value has been found, you can also compute the value of the function at that point.

  • Select learn mode
  • Use A' as label,  2nd   A' 
  • Enter f(x), do not use  =  or  CLR 
  • End f(x) with  INV   SBR 
  • Leave learn mode
  • Select program,  2nd   Pgm   1   6 
  • Define maxmimum number of iterations (optional), press  A 
  • Enter search increment, press  B 
  • Enter starting value x0 and find next critical value by pressing  C . Then press  2nd   A'  to compute f(xcrit) and use  x⇌t  which will display -1 if f(xcrit) is a maximum and 1 if f(xcrit) is a minimum.
  • Use  D  to find next critical value. Or  2nd   D'  to search without limit.

Registers used: 9-13
