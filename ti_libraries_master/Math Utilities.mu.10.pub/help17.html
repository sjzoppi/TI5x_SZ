One of the most powerful and efficient techniques of numerical integration is a systematic procedure called Romberg Integration. This program allows you to choose a desired accuracy for approximation of the integral:

∫ f(x) dx

To run this program, you must enter a subroutine for computing f(x) callable with label  2nd   A' .

  • Select learn mode
  • Use A' as label,  2nd   A' 
  • Enter f(x), do not use  =  or  CLR 
  • End f(x) with  INV   SBR 
  • Leave learn mode
  • Select program,  2nd   Pgm   1   7 
  • Enter lower limit, press  A 
  • Enter upper limit, press  B 
  • Enter accuracy limit, press  C 
  • To evaluate the integral, press  D 

Registers used: 8-18
