SY-07, SY-08 form a group and are to be used together.

The is a two program set which provides a means of balancing a closed traverse by either compass or transit rule. They are designed for use either with Closure (SY-06) or alone.

Given a total distance traversed, the end coordinates of the closure, and the end coordinates of a traverse leg, this program calculates the adjusted (balanced) coordinates. The compass rule, which assumes roughtly equivalent precision in both angular and distance measuremnets.
