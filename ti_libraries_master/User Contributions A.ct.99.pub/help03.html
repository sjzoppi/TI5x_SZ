JS-03 — Limits of functions and infinite series

Given a function f(x), this program computes :

  • lim f(Xo + h), where h -> 0
  • lim f(x), where x -> infinite
  • if of significance, its slanted asymptote

Since the Ti59 has a limited number of significant digits, a Richardson extrapolation is used to increase precision with only a small amount of iterations. For more info, see Richardson extrapolation on Wikipedia.

Given a Un series, it calculates its infinite sum.

Again due to precision issues, an Aitken precision enhancement algorithm has been used. See Aitken's delta squared process  on Wikipedia.

First you need to enter the function in Lbl E into bank 0:

  • Press  2nd   Pgm   0   0 
  • Entrer programming mode, press  LRN 
  • Press  2nd   Lbl   E  and enter f(x) or Un The variable x or n is in the register 07.
  • Always terminate with  =  and then  INV   SBR 
  • Then quit the programming mode by pressing  LRN 
  • Then go back to the limit program, press  2nd   Pgm   0   3 

Limit of f(x) where x tends to xo

  • Actually f(xo+h) is calculated with h becoming smaller every iteration, starting at 0.001 and divided by 10 at each iteration.
  • Five iterations are done in order to have data for calculating a Richardson triangle

Limit of f(x) where x tends to +/- infinite

  • f(x) is calculated with x starting at 1000, at each iteration it is multiplied by 10.
  • Five iterations are done in order to have data for calculating a Richardson triangle

Asymptotes

  • Lim f(x) where x -> Xo may compute a vertical asymptote
  • Lim f(x) where x-> +/- infinite may compute a horizontal asymptote
  • the equation of y = ax + b is calculated
  • where a = lim f(x) ÷ x, with x -> infinite
  • where b = lim f(x) - ax, with x -> infinite

Infinite series

  • the sum of Un is calculated with n starting from 1 to infinite, to find out whether it converges or not
  • Aitken process is done at the end of the summing

Example 1

Lim (x3 - 1) ÷ (x - 1), with x -> 1

Enter routine as described above.

 2nd   Lbl   E 

 (   RCL   0   7   Xy   3   -   1   ) 

 ÷ 

 (   RCL   0   7   -   1   ) 

 =   INV   SBR 

Then:

  • Enter 1, press  A 
  • To compute limite, press  2nd   A' , it displays 2.999998019

Example 3

The curve y = (x2 + 2x - 1) ÷ x has two asymptotes:

  • the Y axis, line x = 0
  • the line y = x + 2

To compute both asymptotes, enter function as above, then:

  • Enter 0, press  A 
  • To compute the limite,  2nd   A' , it displays -14026222.738, it means -Inf, hence the line x=0 is an asymptote.
  • Enter 100, press  C 
  • Then press  2nd   C' , it displays .9999999868
  • Press  R/S  it displays 2.0132049.
  • Press  R/S  it displays -10101010 (end of data)
